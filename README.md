## 🧠 О проекте

Это статичная веб-страница блога **Calm**, сверстанная с нуля.

Проект сделан для практики:
- HTML-разметки
- CSS-стилей
- работы с макетом из Figma
- позиционирования элементов

---

## 🚀 Что реализовано

- Header (логотип + меню)
- Hero-блок (крупный заголовок)
- Секция с карточками (Grid)
- Работа с изображениями
- Базовая структура страницы

---

## 🛠 Технологии

- HTML5
- CSS3 (Flexbox + Grid)
